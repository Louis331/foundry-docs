# Makwright — Project Documentation

> Factory automation game · Godot 4.5 .NET · C#  
> Last updated: June 2026

## Project Overview

Makwright is a 2D top-down factory automation game. The player manually mines raw resources, places machines, and eventually builds automated production pipelines.

**Technology:**
- Godot 4.5 .NET
- C# (primary language)
- `System.Text.Json` for data loading
